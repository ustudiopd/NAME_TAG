### 3. 가상 스크롤링 (`components/VirtualizedProfileList.js`)

- **고정 높이**: `ITEM_HEIGHT = 80px`로 일관된 레이아웃
- **버퍼링**: `BUFFER_SIZE = 5`로 부드러운 스크롤 경험

### 4. 사용자 경험 개선
- **로딩 상태**: 스크롤 중과 페이지 로딩 시 시각적 피드백

- **항목 카운트**: 현재 표시 중인 항목 범위 표시
- **반응형 디자인**: 다양한 화면 크기에서 최적화
